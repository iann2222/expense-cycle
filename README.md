# ExpenseCycle

ExpenseCycle 是一個以「週期性支出 / 訂閱制費用」為核心的個人管理工具，目標是讓你清楚掌握：

- 每個月 / 每年固定會發生哪些支出  
- 接下來什麼時候要繳費  
- 各項訂閱與費用在「年 / 月」角度下的實際負擔  
- 不同類型支出（影音、居住、保險…）的結構與比例  

本專案以 **PWA 網頁 App** 為目標，優先滿足「自己與家人朋友使用」，並保留未來加入雲端同步與帳號系統的擴充彈性。

---

## 🎯 專案目標

- 專注管理「週期性費用 / 訂閱制支出」
- 比一般記帳工具更重視：
  - 週期
  - 到期時間
  - 長期負擔
- 支援「年 / 月統計視角切換」
- 可自訂標籤分類
- 初期不依賴雲端，資料保存在本機裝置
- 架構上可平滑擴充至雲端同步

---

## 🧩 核心概念

每一筆項目都是「會重複發生的費用」，包含：

- 名稱（房租、Netflix、保險…）
- 金額
- 週期（每月 / 每年 / 未來可擴充）
- 可繳日（payable date）
- 截止日（due date）
- 付款方式
- 自訂標籤（可多個）

系統以這些資料為基礎，提供：
- 即將到期項目清單
- 年 / 月統計換算
- 分類彙總視圖

---

## ✅ 已完成的基礎功能（MVP）

### 1. 項目管理
- 新增 / 編輯 / 刪除週期性費用
- 設定週期、可繳日、截止日
- 記錄付款方式
- 自訂標籤

### 2. 檢視與整理
- 列表顯示所有項目
- 依到期日排序
- 依標籤篩選
- 搜尋（名稱 / 標籤 / 備註）
- 依週期、付款方式篩選（規劃中）

### 3. 統計與換算
- 年統計 / 月統計切換
- 年繳項目自動換算成月負擔
- 顯示總固定支出
- 依標籤加總

### 4. 使用體驗
- Material Design 風格介面
- 手機與桌機自適應
- PWA（可加入主畫面，類似 App 使用）

### 5. 資料儲存
- 使用 IndexedDB 儲存在裝置本機
- 重整頁面資料不會消失
- 不需帳號即可使用
- 回收桶機制（30 天後自動清除）
- 匯出 / 匯入備份檔

## 🚀 延伸功能規劃

### 第一階段擴充（個人使用強化）
- 付款歷史紀錄（每次實際付款）
- 更完整的到期狀態標示（可繳中 / 即將到期 / 已過期）
- 更多週期規則（每季、每 N 個月、自訂）

### 第二階段擴充（結構升級）
- 雲端同步
- 帳號登入
- 多裝置共用
- 家庭共享清單
- 權限控制（可檢視 / 可編輯）

### 進階方向（視實際需求）
- 類別占比圖表
- 年度支出趨勢
- 多幣別與匯率
- 本機資料加密 / App 鎖
